      Post on 8月 28 2014

      為什麼昨天只有一道題目呢？對於這件事情我只能說抱歉了……
      昨天我找的是一道 JOI 本選第 5 題……所以其實不是一道非常好想和好寫的題目。當然最後我還是看懂了題解，得出了一個自認為對的解法。
      但是本著負責任的精神，我不敢妄言一個沒有通過題目的解法是對的……到現在我還沒有通過這道題……我會儘快把它寫出來然後放出解法。
      残念です

      今天的第一題是一道簡單的題啦。是來自於 JOI13 本選的 T3 。

      （另外這種蛋糕我吃過呢！很好吃的哦~大家也可以去吃吃看呢。）

      年輪蛋糕

      JOI 君與他的兩個妹妹一同吃年輪蛋糕， 蛋糕分爲 N 塊，第 i 塊的大小為 Ai 。
      這 N 塊蛋糕形成了一個環。第 i 塊蛋糕與 第 i+1 塊蛋糕相鄰，特別地，第 N 塊蛋糕與第 1 塊蛋糕相鄰。
      JOI 君要通過切三刀把蛋糕一分爲三，他只能在塊與塊之間切。
      由於 JOI 君是十分禮讓的人，他會把較大的兩塊蛋糕給他的兩個妹妹。自己吃最小的那一塊。
      但由於他也很喜歡吃蛋糕，所以他希望在此基礎上他能吃到最多的蛋糕。

      3 ≤ N ≤ 100000
      1 ≤ Ai ≤ 1000000000
